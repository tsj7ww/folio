# EDA - DIVISIONS Files 

#### Column Name [IDX] -  Dtype (Head / Tail) 
- **ID** [0] - int64 (1 / 11) 
- **divID** [1] - object (E / U) 
- **lgID** [2] - object (AL / UA) 
- **division** [3] - object (AL East / Sole Division) 
- **active** [4] - object (Y / N)
